🚀 Quick Setup

Deploy AI-powered bot Anda sendiri dalam 10 menit

1

Create Telegram Bot

Chat @BotFather untuk create bot dan dapatkan token

2

Setup Google Sheets

Create Google Sheets + Service Account untuk data storage

3

Get AI API Key

Pilih Groq (FREE), Gemini (FREE), atau OpenAI (PAID) untuk AI features

4

Fork & Deploy

Fork repository ke GitHub dan deploy ke Vercel

5

Set Environment Variables

Add BOT_TOKEN, SHEETS_ID, SERVICE_KEY, dan AI_API_KEY

6

Setup Webhook

Set webhook URL untuk connect Telegram bot ke Vercel
